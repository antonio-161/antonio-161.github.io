Blogicum — учебная блог-платформа

Краткое описание: Проект "Blogicum" представляет собой полнофункциональную учебную блог-платформу, разработанную на фреймворке Django.

Основные возможности:

  • Система авторизации и регистрации пользователей.
  • Создание, редактирование и удаление собственных записей (CRUD-операции).
  • Отображение всех записей, а также фильтрация по автору.
  • Система комментариев к записям.
  • Реализация функционала лайков/дизлайков для записей.
  • Пагинация для удобного просмотра большого количества записей.
  • Валидация форм и обработка ошибок.

Использованные технологии: Python, Django, Django ORM, SQLite (для разработки), HTML, CSS, Bootstrap.

Этот проект демонстрирует навыки работы с бэкендом на Django, проектирование баз данных, а также базовые знания фронтенда для создания интерактивных веб-приложений.

Yatube — социальная сеть для блогеров

Краткое описание: Yatube — это социальная сеть для блогеров, где пользователи могут публиковать записи, подписываться на других авторов и оставлять комментарии.

Основные возможности:

  • Создание и публикация постов.
  • Подписка на других авторов и просмотр их ленты.
  • Система комментариев к записям.
  • Авторизация и управление профилем пользователя.
  • Административная панель для управления контентом.

Использованные технологии: Python, Django, Django ORM, PostgreSQL, HTML, CSS, Bootstrap.

Проект демонстрирует навыки работы с базами данных (PostgreSQL), реализацию сложных связей между моделями, а также разработку многопользовательских веб-приложений.

Kittygram — социальная сеть для любителей котиков

Краткое описание: Kittygram — это социальная сеть, созданная специально для любителей кошек. Здесь пользователи могут делиться фотографиями своих питомцев, оставлять комментарии и взаимодействовать с другими кошководами.

Основные возможности:

  • Публикация фотографий котиков.
  • Лайки и комментарии к публикациям.
  • Просмотр ленты фотографий.
  • Управление профилем пользователя и его публикациями.
  • Возможность подписки на других пользователей.

Использованные технологии: Python, Django, Django REST Framework, PostgreSQL, Docker, Nginx, Gunicorn.

Проект демонстрирует навыки разработки REST API, работу с изображениями, развертывание контейнеризированных приложений и создание полноценного полностекового веб-приложения.

Foodgram — платформа для рецептов

Краткое описание: Foodgram — это веб-платформа для публикации кулинарных рецептов, где пользователи могут делиться своими рецептами, составлять списки покупок и многое другое.

Основные возможности:

  • Публикация, редактирование и удаление рецептов с указанием ингредиентов и шагов приготовления.
  • Добавление рецептов в избранное.
  • Формирование списка покупок на основе выбранных рецептов (с расчетом количества ингредиентов).
  • Система подписки на авторов.
  • Удобный поиск и фильтрация рецептов по тегам и ингредиентам.

Использованные технологии: Python, Django, Django REST Framework, PostgreSQL, Docker, Nginx, Gunicorn.

Проект демонстрирует навыки разработки сложных веб-приложений с учетом взаимодействия пользователей, работы с большим объемом данных и создания гибкого REST API для управления рецептами и списками покупок.
